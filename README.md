## 一个简易的nodejs爬虫项目

## 使用说明
### 基本使用
所有的配置项都在 `webpack.config.js` 文件里，根据需要自行更改：

* 爬取的数据默认存放在项目的`data`文件夹下
* node启动文件：`index.js`
* ssr可视化文件：`views/index.ejs`

## 项目启动方法

`yarn start`

## 环境要求
* windows, linux, macos
* nodejs
* php >= 5.6

### Node.js 安装方法：
#### Windows
https://nodejs.org/en/download/
#### MacOS
`brew install node --with-npm`
